# 腾讯云视频点播 (Tencent VOD) 集成指南

本指南详细说明如何集成腾讯云视频点播服务，重点介绍**服务端直传**和**URL拉取上传**两种模式。

## 1. 准备工作

### 1.1 安装依赖

请确保项目中安装了腾讯云 VOD 和 COS 的 PHP SDK。

```bash
composer require tencentcloud/vod qcloud/cos-sdk-v5
```

### 1.2 配置密钥

在项目根目录的 `.env` 文件中添加以下配置：

```ini
# 腾讯云 VOD 配置
vod_tencent_secret_id  = AKIDxxxxxxxxxxx        # SecretId
vod_tencent_secret_key = z83dxxxxxxxxxxx        # SecretKey
vod_tencent_sub_app_id = 15000xxxxx             # (可选) 子应用 ID
```

## 2. 初始化服务

使用工厂方法获取腾讯云服务实例。

```php
use app\data\service\Vod;

try {
    // 获取服务实例
    $service = Vod::mk(Vod::TENCENT);
} catch (\Exception $e) {
    // 处理初始化异常
    echo $e->getMessage();
}
```

## 3. 核心流程：上传视频

腾讯云 VOD 提供了两种后端上传方式，请根据业务场景选择。

### 方式一：服务端直传 (推荐)

**适用场景**：视频文件存储在您的应用服务器本地，需要上传到 VOD。

**流程说明**：
1.  调用 `uploadVideo` 方法，传入本地文件路径。
2.  服务内部会执行以下步骤：
    *   `ApplyUpload`: 申请上传，获取存储桶、路径及临时凭证。
    *   `COS Client`: 使用临时凭证将文件上传到指定的 COS 存储桶。
    *   `CommitUpload`: 确认上传，通知 VOD 系统生成文件。
3.  上传成功后，返回 `FileId`。

**代码示例**：

```php
try {
    $filePath = '/www/uploads/movie.mp4'; // 本地文件绝对路径
    $title    = '测试视频标题';

    // 执行上传
    $fileId = $service->uploadVideo($filePath, $title);

    echo "上传成功！FileId: " . $fileId;
} catch (\Exception $e) {
    echo "上传失败: " . $e->getMessage();
}
```

---

### 方式二：URL 拉取上传

**适用场景**：视频文件已经存储在 COS 或其他 HTTP 可访问的地址，需要迁移到 VOD。

**流程说明**：
1.  调用 `uploadVideoByUrl` 方法，传入视频 URL。
2.  腾讯云后台会创建一个异步拉取任务。
3.  **注意**：该接口返回的是 `TaskId` (任务ID)，**不是** `FileId`。
4.  您需要通过**回调通知**或**轮询任务状态**接口来获取最终生成的 `FileId`。

**代码示例**：

```php
try {
    $url   = 'https://your-bucket.cos.ap-guangzhou.myqcloud.com/video.mp4';
    $title = 'COS迁移视频';

    // 执行拉取
    $taskId = $service->uploadVideoByUrl($url, $title);

    echo "任务已提交，TaskId: " . $taskId;
    echo "请等待回调通知获取最终 FileId";
} catch (\Exception $e) {
    echo "操作失败: " . $e->getMessage();
}
```

**关于转码**：
*   **默认不转码**：腾讯云 URL 拉取上传后，默认不会触发转码。
*   **如何触发**：需要在腾讯云控制台配置“上传后自动处理”的任务流，或者在上传完成后调用 API 手动发起转码任务。

## 4. 核心流程：获取播放地址

上传成功并完成处理后，可以通过 `FileId` 获取播放地址。

```php
try {
    $fileId = '528589...'; // 之前上传返回的 ID

    $result = $service->getPlayInfo($fileId);

    // 打印播放信息
    $playInfo = $result['playInfo'];
    echo "视频地址: " . $playInfo['url'];
    echo "封面地址: " . $playInfo['cover'];

    // 打印基础信息
    echo "视频标题: " . $result['base']['Title'];

} catch (\Exception $e) {
    echo "获取失败: " . $e->getMessage();
}
```

## 5. 常见问题

1.  **签名有效期**：SDK 内部自动处理签名，无需手动干预。
2.  **子应用 (SubAppId)**：如果您使用了腾讯云 VOD 的子应用功能，请务必在 `.env` 中配置 `vod_tencent_sub_app_id`，否则资源将上传到主应用。
