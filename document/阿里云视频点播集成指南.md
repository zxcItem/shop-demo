# 阿里云视频点播 (Aliyun VOD) 集成指南

本指南详细说明如何集成阿里云视频点播服务，重点介绍**服务端直传**和**URL拉取上传**两种模式。

## 1. 准备工作

### 1.1 安装依赖

请确保项目中安装了阿里云 VOD 和 OSS 的 PHP SDK。

```bash
composer require alibabacloud/vod-20170321 aliyuncs/oss-sdk-php
```

### 1.2 配置密钥

在项目根目录的 `.env` 文件中添加以下配置：

```ini
# 阿里云 VOD 配置
vod_aliyun_access_key = LTAI5t7xxxxxxxx        # AccessKey ID
vod_aliyun_secret_key = dsf78sxxxxxxxxx        # AccessKey Secret
vod_aliyun_endpoint   = vod.cn-shanghai.aliyuncs.com # (可选) 接入点，默认上海
```

## 2. 初始化服务

使用工厂方法获取阿里云服务实例。

```php
use app\data\service\Vod;

try {
    // 获取服务实例
    $service = Vod::mk(Vod::ALIYUN);
} catch (\Exception $e) {
    // 处理初始化异常
    echo $e->getMessage();
}
```

## 3. 核心流程：上传视频

阿里云 VOD 提供了两种后端上传方式，请根据业务场景选择。

### 方式一：服务端直传 (推荐)

**适用场景**：视频文件存储在您的应用服务器本地，需要上传到 VOD。

**流程说明**：
1.  调用 `uploadVideo` 方法，传入本地文件路径。
2.  服务内部会自动申请凭证、初始化 OSS 客户端并将文件推送到云端。
3.  上传成功后，直接返回 `VideoId`。

**代码示例**：

```php
try {
    $filePath = '/www/uploads/movie.mp4'; // 本地文件绝对路径
    $title    = '测试视频标题';

    // 执行上传
    $videoId = $service->uploadVideo($filePath, $title);

    echo "上传成功！VideoId: " . $videoId;
} catch (\Exception $e) {
    echo "上传失败: " . $e->getMessage();
}
```

---

### 方式二：URL 拉取上传

**适用场景**：视频文件已经存储在 OSS 或其他 HTTP 可访问的地址，需要迁移到 VOD。

**流程说明**：
1.  调用 `uploadVideoByUrl` 方法，传入视频 URL。
2.  阿里云后台会创建一个拉取任务。
3.  **注意**：阿里云接口通常会同步返回生成的 `VideoId`，但视频处理（如转码）是异步的。

**代码示例**：

```php
try {
    $url   = 'https://your-bucket.oss-cn-shanghai.aliyuncs.com/video.mp4';
    $title = 'OSS迁移视频';

    // 执行拉取
    $videoId = $service->uploadVideoByUrl($url, $title);

    echo "任务提交成功！VideoId: " . $videoId;
} catch (\Exception $e) {
    echo "操作失败: " . $e->getMessage();
}
```

**关于转码**：
*   拉取成功后，会自动触发阿里云 VOD 的**默认转码模板组**。
*   如果需要指定特定模板组，请在控制台设置默认工作流，或扩展接口代码。

## 4. 核心流程：获取播放地址

上传成功并完成转码后，可以通过 `VideoId` 获取播放地址。

```php
try {
    $videoId = '1b13b7...'; // 之前上传返回的 ID

    $result = $service->getPlayInfo($videoId);

    // 打印播放列表
    foreach ($result['playInfo'] as $playInfo) {
        echo "清晰度: " . $playInfo['definition'] . "\n";
        echo "播放地址: " . $playInfo['url'] . "\n";
    }

    // 打印基础信息
    echo "视频标题: " . $result['base']['Title'];

} catch (\Exception $e) {
    echo "获取失败: " . $e->getMessage();
}
```

## 5. 常见问题

1.  **Endpoint 问题**：默认使用上海节点 (`vod.cn-shanghai.aliyuncs.com`)。如果您的服务开通在其他区域（如北京），请务必在 `.env` 中配置 `vod_aliyun_endpoint`。
2.  **权限问题**：请确保使用的 AccessKey 拥有 `AliyunVODFullAccess` 和 `AliyunOSSFullAccess` 权限。
